# Spectral Metrology and Nanofabrication

This research focuses on the optical characterization and failure analysis of nanofabricated metal-glass structures, utilizing high-sensitivity spectrometry and numerical simulations to validate fabrication quality and sensing performance.

## Sub-100nm Nanodisks Fabrication and

I designed and fabricated gold nanodisk arrays using E-beam lithography at Havard CNS. Also  a metrology pipeline was introduced to characterize nanodisk arrays for localized surface plasmon resonance (LSPR) sensing.

- **System Integration:** Integrated a spectrometer into an upright microscope featuring a custom dark-field illumination system to capture scattering spectra.
- **Refractive Index Sensing:** Validated sensing performance by measuring peak wavelength shifts across varying glycerol concentrations (n=1.33 to 1.44), achieving a high correlation (R2=0.9662) between index changes and spectral response.
- **Numerical Validation:** Successfully correlated experimental dark-field scattering data with FDTD simulations, confirming that far-field scattering peaks accurately reflected near-field electromagnetic enhancements.

## Diagnostic Analysis of Diffractive Nanohole Structures

This project involved the quality control and failure analysis of multi-layer Au/Ti/Quartz diffractive nanoholes.

- **Anomaly Detection:** Identified a performance deviation in outsourced chips that exhibited unexpectedly high transmission in the ~500 nm wavelength range.
- **Failure Analysis via FDTD:** Used FDTD simulations to model various fabrication defects. The analysis proved that an under-etch metal residual layer was responsible for the transmission artifact.
- **Metrology Validation:** Used SEM imaging and experimental spectral data to verify the physical dimensions and stack integrity of the 80 nm Au / 20 nm Ti layers.
